Techflare Nexus — Static Website Demo
-------------------------------------

This project is a single-page static website demo built for the fictional company "Techflare Nexus".

What's included:
- index.html — main page
- styles.css — custom styles and animations
- app.js — small JS for interactivity
- assets/logo.svg — SVG logo
- assets/favicon.svg — favicon
- README.md — this file

How to run locally:
1. Unzip the package.
2. Serve with a static server. Example (Python 3):
   python -m http.server 8000
   then open http://localhost:8000 in your browser.
